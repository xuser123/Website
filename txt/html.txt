# HTML - Hypertext Markup Language

## Basis

<!DOCTYPE html>
<html>
<head>
</head>
<body>
</body>
</html>

## Metadata

<!DOCTYPE html>
<html lang="en-US">
<head>
 <title>Browsertab-Title</title>
 <link rel="stylesheet" type="text/css" href="style.css">
 <meta name="viewport" content="width=device-width, initial-scale=1.0" />
 <meta name="description" content="max150chars">
 <meta name="keywords" content="tags">
 <meta name="author" content="creatorname">
</head>
<body>
</body>
</html>

## Stylings

<!DOCTYPE html>
<html>
<head>
<style>
body {background-color:black;}
h1 {color:red;}
</style>
</head>
<body>
<h1>redonblack</h1>
</body>
</html>
